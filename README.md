GPU Compatibility Matrix 🎮
A comprehensive guide to finding the perfect GPU for your specific hardware setup. Whether you are building a custom gaming rig, upgrading a compact mini-PC, or looking for external GPU (eGPU) solutions, this matrix helps you navigate the complexities of power requirements, physical dimensions, and interface bottlenecks.

📌 Project Overview
Choosing a GPU isn't just about performance; it's about compatibility. This repository provides a structured data set and guide to help users match graphics cards with various systems based on:

Form Factor: Low-profile, single-slot, and ITX-sized cards.

Power Constraints: TDP limits and required PCIe power connectors.

Thermal Limitations: Compatibility with small-form-factor (SFF) cases.

Interface: PCIe 4.0/5.0 lane availability and eGPU (Thunderbolt/OCuLink) support.
How to Use This Repo
Search by Device: Look through the /devices folder to find your specific model or form factor.

Check the Specs: Review the compatibility_list.md for a quick glance at verified pairings.

Contribute: Have a setup that works? Submit a Pull Request with your specs to help the community!

🛠️ Contribution Guide
We love community input! If you want to add a verified GPU/System pairing:

Fork the repository.

Add your entry to the relevant .csv or .md file.

Include your system specs (CPU, PSU Wattage, Case).

Open a Pull Request.
